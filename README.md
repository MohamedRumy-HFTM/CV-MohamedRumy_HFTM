## 🚀 Installation & Setup

### 🌐 Live Website
**URL:** https://mohamedrumy.ch

### Lokale Entwicklung
1. Repository klonen:
   ```bash
   git clone [repository-url]
   cd CV-MohamedRumy_HFTM
   ```

2. Lokalen Server starten:
   ```bash
   # Mit Python 3
   python -m http.server 8000
   
   # Mit Node.js
   npx serve .
   
   # Mit PHP
   php -S localhost:8000
   ```

3. Browser öffnen: `http://localhost:8000`
